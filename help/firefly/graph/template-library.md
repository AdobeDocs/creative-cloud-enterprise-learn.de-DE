# &#x200B;5. Vorlagenbibliothek

Ein Schnellreferenz-Index von Firefly-Diagramm-Vorlagen, der nach den einzelnen erstellten oder ausgeführten Vorgängen organisiert ist. Jedes Beispiel ist der Ausgangspunkt. Tausche dein Branding, Produkt und deine Eingabeaufforderungen aus, bevor du eine Vorlage in der Produktion verwendest.

## Bilderstellung und -stil

| Diagrammvorlage | Beschreibung | [!BADGE Anwendungsfälle]{type=Informative tooltip="Anwendungsfälle"} |
|---|---|---|
| [**Erste Schritte - Ein Bild generieren**](/help/firefly/graph/templates/get-started-gen-image.md) | Diese Vorlage ist ein einfaches Diagramm: einen Aufforderungsknoten in einen Erzeugungsknoten in einer Ausgabe. | Einzelhandel, Gesundheit, Bildung |
| [**Konsistente Zeichengenerierung**](/help/firefly/graph/templates/character-gen.md) | In dieser Graphvorlage laden Sie ein Referenzbild einer Figur und tauschen dann die Szene oder die Posenaufforderung für jede neue Aufnahme aus. | Reisen, Einzelhandel, Bildung |
| [**Stilextraktion**](/help/firefly/graph/templates/style-extraction.md) | Diese Vorlage gibt das Referenzbild wieder, um Farbe, Licht und Struktur zu extrahieren. | Reisen, Einzelhandel, Getränke |
| [**Sunset Vibes**](/help/firefly/graph/templates/sunset-vibes.md) | In dieser Graphvorlage können Sie ein 3D-Typografiebild aus einer Textaufforderung erstellen. | Reisen, Getränke, Einzelhandel |
