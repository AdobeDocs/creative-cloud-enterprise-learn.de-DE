# &#x200B;5. Vorlagenbibliothek

Ein Schnellreferenz-Index von Firefly-Diagramm-Vorlagen, der nach den einzelnen erstellten oder ausgeführten Vorgängen organisiert ist. Jedes Beispiel ist der Ausgangspunkt. Tausche dein Branding, Produkt und deine Eingabeaufforderungen aus, bevor du eine Vorlage in der Produktion verwendest.

## Bilderstellung und -stil

| Diagrammvorlage | Beschreibung | [!BADGE Anwendungsfälle]{type=Informative tooltip="Anwendungsfälle"} |
|---|---|---|
| [**Erste Schritte - Ein Bild generieren**](/help/firefly/graph/templates/get-started-gen-image.md) | Diese Vorlage ist ein einfaches Diagramm: einen Aufforderungsknoten in einen Erzeugungsknoten in einer Ausgabe. Verwende es als erste Vorlage, die bei einem neuen Anwender geöffnet wird. | Einzelhandel, Gesundheit, Bildung |
| [**Konsistente Zeichengenerierung**](/help/firefly/graph/templates/character-gen.md) | In dieser Graphvorlage laden Sie ein Referenzbild einer Figur und tauschen dann die Szene oder die Posenaufforderung für jede neue Aufnahme aus. Die Zeichenreferenz bleibt gesperrt, während sich die umgebende Szene ändert. | Reisen, Einzelhandel, Bildung |
| [**Stilextraktion**](/help/firefly/graph/templates/style-extraction.md) | Diese Vorlage gibt das Referenzbild wieder, um Farbe, Licht und Struktur zu extrahieren. Sie können diese Behandlung auf jedes neue Bild anwenden, das über denselben Graphen läuft. | Reisen, Einzelhandel, Getränke |
| [**Sunset Vibes**](/help/firefly/graph/templates/sunset-vibes.md) | In dieser Graphvorlage können Sie ein 3D-Typografiebild aus einer Textaufforderung erstellen. Die Vorlage sorgt automatisch für Platzierung und Farbbalance. | Reisen, Getränke, Einzelhandel |

## Segmentierung und Compositing

| Diagrammvorlage | Beschreibung | [!BADGE Anwendungsfälle]{type=Informative tooltip="Anwendungsfälle"} |
|---|---|---|
| [**Erste Schritte - Segmentieren eines Bildes**](/help/firefly/graph/templates/get-started-segment-image.md) | In dieser Graphvorlage laden Sie ein beliebiges Quellbild und führen den Segmentierungsknoten aus, um das Motiv von seinem Hintergrund zu isolieren. Stelle einen Ersatzknoten für den Hintergrund zusammen, um einen sauberen Ausschnitt zu erhalten. | Gesundheit, Einzelhandel, Automobil |
| [**Ebenen kombinieren und überblenden**](/help/firefly/graph/templates/composite-blend-layers.md) | In dieser Graphvorlage stapelst du einen Produktausschnitt und eine Hintergrundszene als separate Ebeneneingaben. Passe Mischmodus und Beleuchtungsknoten an, bis sich die Komposition als eine Aufnahme ausliest. | Getränke, Einzelhandel, Reisen |
| [**Selektive Farbkorrektur**](/help/firefly/graph/templates/selective-color-correction.md) | In dieser Graphvorlage maskieren Sie einen bestimmten Bereich, der korrigiert werden muss, und legen die Zielfarbe nur für diesen Knoten fest. Der Rest des Bildes wird unberührt durch den Graphen geleitet. | Telekommunikation, Einzelhandel, Finanzen |

|---|---|---|

## 3D/Character Animator

| Diagrammvorlage | Beschreibung | [!BADGE Anwendungsfälle]{type=Informative tooltip="Anwendungsfälle"} |
|---|---|---|
| [**Echtzeit-Shader**](/help/firefly/graph/templates/real-time-shaders.md) | In dieser Graphvorlage beginnen Sie mit einem Bild und wenden drei verschiedene benutzerdefinierte Shader an und zeigen das Ergebnis in Echtzeit an. Passen Sie die Shader-Parameter direkt auf dem Knoten an, anstatt das Rendering von Grund auf neu durchzuführen. | Technologie, Automobil, Einzelhandel |
| [**Generierung des Zeichenmodells**](/help/firefly/graph/templates/character-model-generation.md) | Mit dieser Vorlage erstellen Sie einen animierten 3D-Stil für eine Illustration. Die Vorlage generiert einen grundlegenden 3D-Modell-Pass, der einem Modellierer zur Bereinigung übergeben wird, anstatt mit einer leeren Szene zu beginnen. | Outdoors, Tech, Education. |

Bei der Lizenzierung oder beim Review von Produkten gibt es Eckpunkte. | Einzelhandel, Getränke, Unterhaltung |
| [**Umwandlung von Skizzen in 3D**](/help/firefly/graph/templates/sketch-to-3d.md) | In dieser Graphvorlage verwandeln Sie eine Skizze in eine 3D-Figur. ## Produkt- und Marken-Mockups

| Diagrammvorlage | Beschreibung | [!BADGE Anwendungsfälle]{type=Informative tooltip="Anwendungsfälle"} |
|---|---|---|
| [**Branding-Visualisierung**](/help/firefly/graph/templates/branding-visualization.md) | In dieser Vorlage lernst du, wie du ein Logo oder eine Marke in Produktszenen visualisieren kannst. Definiere Markenrichtlinien oder eine Logo- oder Farbpalette. Das Diagramm gibt die statische Key Art und einen kurzen Bewegungs-Pass in einem Durchgang aus, sodass beide Formate visuell ausgerichtet bleiben. | Technologie, Getränke, Finanzen |
| [**Mockup des Markenprodukts**](/help/firefly/graph/templates/brand-product-mockup.md) | Erfahre, wie du dein Produkt in verschiedenen Szenen visualisieren kannst. Du legst ein Produkt-Rendering oder Foto im Mockup-Knoten ab. Das Diagramm platziert es innerhalb einer Szene mit einem vollständigen Branding, wobei Beleuchtung und Schatten automatisch an diese Szene angepasst werden. | Einzelhandel, Getränke, Technik |
| [**Editorial-Fotoshooting**](/help/firefly/graph/templates/editorial-photoshoot.md) | In dieser Graphvorlage laden Sie eine Modellreferenz und tauschen die Kleidungseingaben für jeden neuen Look aus. Posen und Beleuchtungsknoten bleiben im gesamten Set gesperrt, um ein einheitliches redaktionelles Erscheinungsbild zu erzielen. | Einzelhandel, Schönheit, Außenbereich |
| [**Fotografiestudio**](/help/firefly/graph/templates/photography-studio.md) | In dieser Graphvorlage platzierst du ein Produkt-Rendering auf dem Studiohintergrund und passt die Beleuchtung an, bis das Ergebnis wie eine echte Studioaufnahme aussieht. | Getränke, Technologie, Einzelhandel |
| [**Aufkleber auf Oberflächen anwenden**](/help/firefly/graph/templates/decal-to-surfaces.md) | In dieser Graphvorlage laden Sie das Basisprodukt-Mockup und das Aufkleber- oder Logo-Element als separate Eingaben. Die Schablone passt den Aufkleber an die Oberflächengeometrie an, sodass er den Konturen korrekt folgt. | Outdoors, Automotive, Retail |
