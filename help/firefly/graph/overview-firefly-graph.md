# Erste Schritte mit Firefly Graph

Wenn du schon einmal ein generatives KI-Tool verwendet hast, bist du wahrscheinlich daran gewöhnt, eine Eingabeaufforderung einzugeben, ein Ergebnis zu erhalten und beim nächsten Mal von vorne anzufangen, wenn du etwas Ähnliches brauchst. Firefly Graph funktioniert anders. Anstelle einer einzelnen Eingabeaufforderung erstellen Sie ein Diagramm: einen visuellen, schrittweisen Arbeitsablauf, bei dem jede Eingabe, Transformation und Ausgabe verbunden und zusammen gespeichert wird. Diese Seite führt Sie durch alle notwendigen Schritte, um Ihr erstes Diagramm zu erstellen: die grundlegenden Konzepte kennen, einen ersten Arbeitsablauf Schritt für Schritt sowie eine Bibliothek gebrauchsfertiger Vorlagen, die du direkt öffnen und anpassen kannst.

>[!TIP]
>
>Stelle es dir wie ein Rezept vor, anstatt wie eine einzige Mahlzeit. Sobald du ein Diagramm erstellt hast, kannst du es übergeben, auf neuen Eingängen erneut ausführen oder einen Schritt ändern, sodass alles Downstream automatisch aktualisiert wird.
