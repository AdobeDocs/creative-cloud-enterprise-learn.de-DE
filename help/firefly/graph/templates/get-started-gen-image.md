# Erste Schritte - Bild generieren

Lernen Sie die Grundlagen mit der einfachsten möglichen Grafik: einen Aufforderungsknoten in einen Erzeugungsknoten in einer Ausgabe. [Öffnen Sie Erste Schritte - Generieren Sie eine Bildvorlage &#x200B;](https://firefly.adobe.com/graph/edit/id/urn:aaid:sc:VA6C2:f556b557-d31b-4659-b9d1-358ed567c61d).

>[!TIP]
>
>**Bevor Sie beginnen** - Passen Sie diese Vorlage immer an Ihr eigenes Branding, Produkt und Ihren eigenen Workflow an. Tauschen Sie Ihre Referenzbilder, Eingabeaufforderungen und Texte ein, bevor Sie eine Ausgabe verwenden.

![General image](../../assets/get-started-gen-image.png){align="center"}

| Typ | Ausweis |
|---|---|
| Informativ (Standard) | [!BADGE Anwendungsfälle]{type=Informative tooltip="Anwendungsfälle"} |

* **Einzelhandel** - Generieren Sie ein erstes Produktheldenbild aus einem Briefing, um den grundlegenden Knotenfluss zu erlernen, bevor Sie ein echtes Kampagnen-Asset berühren.
* **Gesundheit** - Testen Sie den einfachsten Bildgenerierungsfluss auf einem Platzhalterprodukt, bevor Sie auf einen vollständigen Inhaltskalender skalieren.
* **Education** - Erstellen Sie ein erstes Beispielbild, um das Diagramm neuen Teammitgliedern zu zeigen, bevor Sie echte Projektarbeit zuweisen.
