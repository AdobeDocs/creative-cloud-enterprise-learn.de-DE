# Porträtfotos - Generationen

Lerne, wie du eine Reihe von Business-Porträts aufeinander abstimmen kannst. Das Diagramm normalisiert Beleuchtung, Hintergrund,
und über das gesamte Set in einem Durchgang zuschneiden. [Vorlage für die Generierung von Headshots öffnen](https://firefly.adobe.com/graph/edit/id/urn:aaid:sc:US:5da3f95f-63e5-5335-9e10-58cfadd7ad3f).

>[!TIP]
>
>**Bevor Sie beginnen** - Um optimale Ergebnisse zu erzielen, passen Sie diese Vorlage an Ihr eigenes Branding, Produkt und Ihren eigenen Workflow an. Tauschen Sie Ihre Referenzbilder, Eingabeaufforderungen und Texte ein, bevor Sie eine Ausgabe verwenden.

![Generierung von Porträtfotos](../../assets/headshots-generation.png){align="center"}

[!BADGE Anwendungsfälle]{type=Informative tooltip="Anwendungsfälle"}

* **Tech** - Generieren Sie ein konsistentes Porträtsatz für ein aktualisiertes Mitarbeiterverzeichnis, bevor Sie alle Hände bedienen, ohne einen Fotografen für jeden neuen Mitarbeiter zu planen.
* **Finanzen** - Normalisieren Sie die Porträtfotos in einem Beraterteam für eine Besprechungsseite des Teams.
* **Gesundheit** - Standardisieren Sie die Porträtfotos der Mitarbeiter über mehrere Klinikstandorte hinweg, um einen einheitlichen Website-Look zu erzielen.
